Esta página utiliza fontes que foram importadas da Web direto para o código. O site Google Fonts possui em torno de 2.000 fontes disponíveis gratuitamente para uso, com ele é possível selecionar e importar fontes de acordo com sua família tipográfica, anatomia e categoria.

A fonte selecionada para os títulos foi a "Kaushan Script", uma fonta não serifada que pertence a família tipográfica Script/Manuscrita, também conhecidas como "handwriting". Fontes handwriting possuem uma anatomia feita para simular a escrita humana.

Para os parágrafos foi escolhida a fonte "Work Sans", uma fonte pertencente a família tipográfica Sans-Serif, ou seja, não possui serifa. Fontes não serifadas possuem uma antomia ótima para textos em Web, pois transmitem uma sensação de limpeza e organização para que lê.
