↓ Tags que estão obsoletas em HTML5 ↓

OBS: As tags a seguir ainda existem e funcionam, porém,
é recomendado que não sejam mais utilizadas em HTML5 moderno!

Tags de apresentação visuais (Substituidas por CSS):

<font>   → Em CSS →   (color, font-family)
<center>   →   text-align: center
<big>   →   font-size
<strike>   →   text-decoration: line-through
<tt>   →   font-family: monospace

Estrutura antiga / desnecessária:

<frame>
<frameset>
<noframes>

Substituído por: <iframe>

Semântica ruim ou substituída:

<acronym>   → use   → <abbr>
<applet>   → use   → <object> ou JS moderno
<dir>   → use   → <ul>
<isindex>   → removido completamente

Multimídia antiga:

<basefont>
<bgsound>
<marquee>

Substituido por: CSS animations / JavaScript
